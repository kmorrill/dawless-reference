# Teenage Engineering TX-6

## Overview
- **Type**: Ultra-portable 6-channel stereo mixer + audio interface
- **Channels**: 6 stereo input channels
- **Effects**: Built-in FX per channel
- **Audio Interface**: USB class-compliant, multi-channel

## Connectivity
| Interface | Direction | Details |
|-----------|-----------|---------|
| USB-C | In/Out | Audio interface (multi-channel), MIDI, USB host mode |
| 3.5mm inputs (x6) | In | Stereo line in per channel |
| 3.5mm output | Out | Stereo mix |
| 3.5mm headphone | Out | Stereo, independent level |
| Bluetooth | In/Out | Audio streaming |

### USB Host Mode
The TX-6 can act as a **USB host** — connect class-compliant USB MIDI/audio devices directly without a computer. This makes it a hub for dawless setups.

## MIDI
- USB MIDI (class-compliant)
- CC control of mixer parameters (volume, pan, FX sends)
- Can route MIDI between connected USB devices in host mode

## Audio Interface
- Multi-channel USB audio (class-compliant)
- Individual channel outputs available over USB
- Works with iOS, macOS, Windows, Linux

## Limitations
- Small physical size makes knobs cramped
- Bluetooth audio adds latency
- Limited EQ (no parametric EQ per channel)
- Battery life varies with USB host power draw
