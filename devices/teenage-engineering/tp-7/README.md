# Teenage Engineering TP-7

## Overview
- **Type**: Portable field recorder / tape machine
- **Recording**: Stereo, high-quality ADC
- **Storage**: Internal + USB transfer
- **Design**: Reel-to-reel aesthetic with physical tape transport controls

## Connectivity
| Interface | Direction | Details |
|-----------|-----------|---------|
| USB-C | Out | File transfer, charging |
| Built-in mic | In | Stereo condenser |
| 3.5mm line in | In | Stereo |
| 3.5mm line out | Out | Stereo |
| Built-in speaker | Out | Mono |

## MIDI
None. The TP-7 is a purely audio device with no MIDI capability.

## Audio Features
- Variable speed playback
- Built-in effects (tape-style)
- Automatic recording modes
- USB file transfer for recorded audio

## Limitations
- No MIDI sync or control
- No audio interface mode over USB
- Internal storage only (no SD card)
