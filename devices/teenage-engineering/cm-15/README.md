# Teenage Engineering CM-15

## Overview
- **Type**: Submixer — 4 stereo inputs to 1 stereo output
- **Inputs**: 4x 3.5mm stereo
- **Output**: 1x 3.5mm stereo
- **Power**: USB-C (power only)

## Connectivity
| Interface | Direction | Details |
|-----------|-----------|---------|
| 3.5mm inputs (x4) | In | Stereo line in |
| 3.5mm output | Out | Stereo mix |
| USB-C | — | Power only |

## MIDI
None. The CM-15 is a passive analog mixer with no digital connectivity.

## Use Case
Sum multiple TE devices (Pocket Operators, OP-Z line out, etc.) into a single stereo output. No level control per channel — all inputs mixed at unity gain.

## Limitations
- No per-channel volume control
- No effects
- No MIDI
- No audio interface capability
- Passive mixing only
